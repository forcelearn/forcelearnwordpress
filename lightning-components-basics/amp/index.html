Today, we are starting about Salesforce lightning component Basics and it usage

what is lightning component ?

Lightning Components is a UI framework which helps to develop web applications for mobile and desktop compatible. It is designed for to create single page applications with dynamic and responsive for salesforce force.com applications and it uses apex on server side and JavaScript on client side.
