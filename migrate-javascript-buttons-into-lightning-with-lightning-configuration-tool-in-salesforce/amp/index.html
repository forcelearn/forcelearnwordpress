Today we discuss about Salesforce Lightning Configuration tool Looking for lightning migration … stuck to convert Javascript button into lightning. Salesforce launched a tool named as Lightning Experience Configuration Converter , It helps us to convert custom JavaScript buttons into lightning

As you aware of that JavaScript buttons and URL hack will now work in lightning Experience. In order to achieve your requirements designed on JavaScript buttons, you need to design lightning component  and call that button using Lightning Quick Action.

In order to reduce lightning migration efforts salesforce launched Lightning experience configuration tool.

Here i have taken an use case. I have design a button on Account Layout, onclick on button it will display alert to user with account name is work designed based on JavaScript button.

Today we discuss about Salesforce Lightning Configuration tool Looking for lightning migration … stuck to convert Javascript button into lightning.

Login to https://lightning-configuration.salesforce.com/

and login into your salesforce sandbox/dev org and check for your button your looking for lightning migration.

I have take here Account Name as sample javaScript button which provide account name in alert. Iam trying to migrate these java script button into lightning

click on that button and select convert and deploy  button will convert in few seconds

Navigate to converted buttons you will find converted button and which contains a lightning component designed for button

login into your org and add into button on page layout and  test it in lightning version. You have successfully migrated javascript button into lightning with help of Lightning Experience configuration tool.
